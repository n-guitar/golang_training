- 1.2 コマンドライン引数
    Goでは全てのインデクス指定は半開区間であり、最初のインデックスを含み最後のインデックスを含みません。

    for文 笑楽可能であるinitialization文

    ブランク識別子
    func main() {
	    s, sep := "", ""
	    for _, arg := range os.Args[1:] {
		    s += sep + arg
		    sep = " "
	    }
	    fmt.Println(s)
    }
